Sorting viaSockets

About this page

This is a tool to interact with a server through vectors and sorting methods. Check the options below and click on any of them to start using it. This tool was possible thanks to Flask and Socket.IO.

Tools

Select one option:

Merge Sort

Heap Sort

Quick Sort

Vector to use

You can introduce the vector yourself, or you generate a random one as well. Note that despite the size of the vector field, the vector can be the size you want (in case of generated ones, only < 7 digits).

Introduce the vector numbers separated by commas like this: 4,32,7,3,1

Sort

Once filled the fields above, click on the button 'Sort' below.
